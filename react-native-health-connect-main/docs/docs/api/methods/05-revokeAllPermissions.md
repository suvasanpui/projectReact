# `revokeAllPermissions`

Revokes all previously granted permissions by the user to the calling app.

# Method

```ts
revokeAllPermissions(): Promise<void>
```

# Example

```ts
import { revokeAllPermissions } from 'react-native-health-connect';

// ...
revokeAllPermissions();
```
